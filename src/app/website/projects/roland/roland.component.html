Industry: Commercial Printing

Roland dG connect designer

UX/UI DESIGN | WEBSITE DESIGN

Roland DGA, the American subsidiary of Japan's historic Roland DG, is a leader in advanced digital printing solutions—from UV printers to dental milling machines and texturing systems—with tools like DIMENSE™ and VersaOBJECT™ for applications ranging from signage to décor and the dental industry.

The Challenge

The main challenge in designing this landing page was to introduce a new software tool (Roland DG Connect Designer) into a generally hardware-focused ecosystem in a way that was clear, engaging, and accessible to users of all skill levels.

The goal was to communicate that this was an all-in-one, intuitive, and professional solution without creating any friction in the user experience. This involved addressing three key challenges from a UX/UI perspective:

  • Simplify the technical aspects without losing professional value: the software offers process automation, preconfigured files, and asset libraries. Translate these technical functionalities into understandable visual and textual language.
  • Prioritize information with clarity and consistency: prioritize a strong core message, using visual sections to highlight specific benefits, such as ease of use, print-ready file generation, and advanced compatibility.
  • Connect emotionally through visual design: displaying images and mockups will reflect real-life usage scenarios with a modern aesthetic, showcasing the software interface integrated with the visual content.

    • Design Process

    From the wireframe phase, the focus was on clearly communicating the value proposition, guiding the user through the information flow, and facilitating conversion with clear calls to action.

    Clarity and Purpose

  • Clearly communicate the benefits of the software.
  • Demonstrate the ease of the workflow with the DIMENSE module.
  • Build confidence in technical compatibility.
  • Encourage free software downloads as a primary conversion.

    • UX Decision:Complex graphics were avoided at this stage to focus on communicating the proposal and allowing for visual flexibility in later UI.

    Principles Applied

    Clear Visual Hierarchy: the typography is carefully structured. The main title uses visual weight and italic style to highlight the key message. The subtitle maintains an optimal length for quick scanning, and the CTA is well separated and accessible.

    Modular and Responsive Design: the two-column layout (text + visual) allows for fluid adaptation to mobile devices. The information cards at the bottom are organized in a consistent grid, easily adaptable to responsive formats (1-2 columns on mobile).

    Functional Visual Support: the images show the software in use, reinforcing the context without requiring additional explanation. This improves product understanding in a visual and direct way.

    Focused CTA: the primary button (“Get Started for Free”) is isolated and strategically positioned to capture attention after reading the main content. It is designed with a clean, accessible style and without close visual competition.

    UI Summary - Feature Cards + Compatibility Section

    This screen corresponds to the middle section of the landing page and its main objective is to communicate key benefits of the software in a clear, scannable, and visually appealing way.

    Card-based layout: modular components in the form of cards were used to divide and highlight the product's main features. This pattern allows for quick reading and improves content comprehension.

    Visual consistency and graphic rhythm: all cards maintain consistent proportions, styles, and spacing. Images are aligned at the top, followed by a block of informational text and a capitalized heading for easy scanning.

    Functional and contextual visuals: each image within the cards reinforces the textual content by showing software or hardware in use. This provides visual context without relying exclusively on text.

    DIMENSE module

    This section is designed to highlight the DIMENSE module as an extension that adds texture and depth to printed designs.

    From a UI perspective, it combines a clear split-screen visual structure with informative text on the left and a functional image on the right. The goal is to communicate the added value of the add-on quickly, visually, and directly, reinforcing the seamless integration with Roland DG Connect Designer software. Prioritize clarity, tangible benefits, and immediate action through a prominent CTA.

    from a uI perspective

    I used a clear, split design that combines attractive graphic elements on the left with a numbered step-by-step guide on the right, improving comprehension and reducing cognitive load. The goal is to convey simplicity and control, showing how the user can easily move from a flat design to a dimensional print without additional tools, in just five steps. This section is designed to reinforce usability and efficiency in the creative process.

    Responsive Design

    Design the mobile version of the landing page with a clear focus on readability, visual hierarchy, and immediate action.

    Present the sections in well-spaced vertical blocks, with scalable typography, prominent call-to-action buttons, and strategic use of color to highlight key messages. Adapt the images without cluttering the space, keeping the focus on the value proposition. The experience is optimized so the user understands the main benefit quickly and frictionlessly, even on small screens.

    Results Obtained

    Increased conversion: the landing page's clear and straightforward design, focused on benefits and with prominent calls to action, made it easier for users to take the next step (registration, free trial, or sales contact).

    Improved product communication: a technical product was translated into a value proposition that was easy to understand for non-specialized users, eliminating barriers to entry.

    Brand positioning as a comprehensive solution: the interface conveys trust, professionalism, and an all-in-one experience, reinforcing Roland's positioning as a software and hardware-integrated solution.

    Lessons Learned

    This project taught me that good design not only solves visual problems, but also boosts commercial results when properly aligned with business objectives: more leads, better product understanding, and higher purchase intent.
